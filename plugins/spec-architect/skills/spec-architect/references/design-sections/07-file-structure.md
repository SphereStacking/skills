# 7. ファイル構成

**ディレクトリツリー:**

```
{project-root}/
├── {dir1}/
│   ├── {file1}        # {説明}
│   └── {file2}        # {説明}
├── {dir2}/
│   ├── {subdir}/
│   │   └── {file3}    # {説明}
│   └── {file4}        # {説明}
└── {config-file}       # {説明}
```

**命名規則:**
