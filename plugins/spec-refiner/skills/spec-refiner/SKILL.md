# ヒアリング・言語化ツール

`/spec-refiner` コマンドで開始。2つのモード:

- **START**: `/spec-refiner start "認証システム"` — 新しいヒアリングを開始
- **CONTINUE**: `/spec-refiner continue` — 既存のヒアリングを再開

## 核心的なコンセプト

- **ヒアリングに徹する**: 完成度・整合性・網羅性は一切チェックしない。ユーザーの言葉をそのまま拾う
- **終了はユーザーが決める**: ユーザーが明確に「終わり」「完了」「ここまで」等と言うまで、絶対に終了を提案しない
- **言語化の支援**: ユーザーの曖昧な表現を明確な言葉に変換し、「こういうことですか？」と確認する
- **記録の蓄積**: 対話の中で出てきた情報をすべて記録し、失わない
- **自然な対話**: フェーズやチェックリストで対話をコントロールしない。ユーザーの話の流れに沿う
- **プロジェクト管理**: `.claude/spec-refiner/{NNN}-{slug}/` 配下にプロジェクトごとに独立管理
